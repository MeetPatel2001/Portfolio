Projects

1. Mercedes-Benz

Reduce the time a Mercedes-Benz car spends on the test bench.

Mercedes-Benz has a lot of features. So, it’s impossible to optimize the testing time for a car without a powerful algorithm. There are 376 potential features a Mercedes-Benz can have according to the dataset. It includes the custom features selected by the user at the time of ordering a car.

Tools/Concepts used: Python, random forest, xgboost, neural net, and principal component analysis.

Libraries used: Pandas, Numpy, Sklearn, TensorFlow, Matplotlib and Seaborn.

2. Global Temperature Shiny App

App link- https://meetpatel.shinyapps.io/Global_Temperature/

Global temperature shiny app created for BAS 475's final.

Tools/Concepts used: R, R-Shiny, Time-series, Arima, ETS, STL, Simple models.

Libraries used: fpp3, plotly, ggplot2, shiny, and urca.

3. Consumer Product Safety Commission (CPSC)

Goal: Find 10 lithium ion based products to avoid manufacturing

Project Design Report

I analyzed all 'Thermal Fire' incidents using US's CPSC data for the year 2023.

Tools/Concepts used: Python, SPSS, JASP, Orange3, PerplexityAI, and AnythingLLM

Libraries used: Pandas, Numpy, NLTK, and Matplotlib.

4. House Prices Advanced Regression

A fun kaggle project to practice predictive models.
